# TravelEats - 여행 식당 추천 시스템

그룹 여행에서 모든 멤버가 만족할 수 있는 식당을 AI가 추천해주는 웹 애플리케이션입니다.

## 🎯 주요 기능

### 1. 사용자 인증
- **회원가입**: 아이디, 비밀번호, 닉네임으로 가입
- **로그인**: 간편한 로그인 시스템
- **고유번호(PID)**: 각 사용자에게 자동으로 부여

### 2. 그룹 관리
- **그룹 생성**: 여행 그룹을 만들고 6자리 코드 발급
- **그룹 참여**: 코드를 입력하여 그룹에 참여
- **멤버 관리**: 그룹 멤버 목록 확인

### 3. 여행 계획
- **여행지 설정**: 목적지와 지역 선택
- **기간 설정**: 1-10일 여행 기간 설정
- **예산 설정**: 1인당 평균 식사 예산 설정

### 4. 음식 선호도 입력
- **좋아하는 음식**: 한식, 중식, 일식, 양식 등 카테고리 선택
- **싫어하는 음식**: 선호하지 않는 카테고리 선택
- **못 먹는 음식**: 알레르기, 금기 음식 선택 (추천에서 완전 제외)
- **선호 키워드**: 매운맛, 해산물, 육류 등 세부 선호도
- **예산 범위**: 개인별 가격대 설정

### 5. AI 기반 식당 추천
- **합의 점수 계산**: 모든 멤버의 선호도를 종합하여 점수 산출
- **정렬 시스템**: 합의 점수 순으로 식당 정렬
- **필터링**: 최소 합의 점수로 필터링 가능
- **상세 분석**: 멤버별 선호도 분석 제공

### 6. 마이페이지
- **개인정보 확인**: 사용자 정보 및 통계
- **선호도 관리**: 음식 선호도 수정
- **그룹 목록**: 참여 중인 그룹 확인

## 🔧 기술 스택

- **React Router v6**: 라우팅

## 🧮 추천 알고리즘 로직

### 개인 선호도 점수 계산
- **기본 점수**: 50점
- **좋아하는 카테고리**: +30점
- **싫어하는 카테고리**: -40점
- **못 먹는 음식 포함**: -50점 (치명적)
- **싫어하는 키워드**: -20점
- **좋아하는 키워드**: +15점
- **예산 범위 일치**: ±10점
- **최종 점수**: 0-100점 범위로 제한

### 그룹 합의 점수 계산
- 모든 멤버의 개인 점수 평균
- 70점 이상: 좋아하는 멤버
- 30점 미만: 싫어하는 멤버
- 그 외: 중립 멤버

## ⚠️ 주의사항

- 이 프로젝트는 Google Places API 대신 Mock 데이터를 사용합니다
- 실제 서비스 적용 시 백엔드 API 연동이 필요합니다
- LocalStorage를 사용하므로 브라우저를 비우면 데이터가 삭제됩니다
- 실제 지도 기능은 구현되어 있지 않습니다 (위치 정보만 표시)
